Projects

Beat That!

"Beat That!" is a solo project that was created after two weeks of learning the fundamentals. It is a quick dice game in which the goal is to roll the dice and play the highest number that can be created to beat the CPU.

Technologies used: javascript, jquery, and DOM manipulation. Feel free to play the game here.

Farm-to-table

"Farm-to-table" is a partner project with one other developer that utilizes the MEN stack. The idea behind this project was to connect customers to local farms in order to purchase fresh produce. My contributions include stylizing of client-side interface and user authentication.

Technologies used: javascript, mongodb, mongoose, express.js, node.js, html/css
