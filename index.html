Projects

Keeb Dreams

Dream Keebs is a project that I hope to develop into a powerful tool for both new and old keyboard enthusiasts. It allows the user to sign in and keep track of the keyboards they already have and the ones they would love to create. The user can also keep track of any interest checks or group buys they are interested in. While the parts in the database are currently bare, it aims to be a community-driven application where the users can contribute by adding various parts with their description. Please note that the login and signup functionality still needs work!
